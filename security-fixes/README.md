## 快速开始

### 方法一：使用主修复脚本（推荐）

```bash
# 进入项目根目录
cd /path/to/your/project

# 运行主修复脚本
node security-fixes/scripts/run-all-fixes.js
```

主修复脚本提供交互式界面，可以：

- 选择执行特定修复或全部修复
- 自动创建系统备份
- 按优先级顺序执行修复
- 生成综合修复报告

### 方法二：单独执行修复脚本

```bash
# JWT安全修复
node security-fixes/scripts/fix-jwt-security.js

# 敏感数据加密
node security-fixes/scripts/encrypt-sensitive-data.js

# CORS配置修复
node security-fixes/scripts/update-cors-config.js

# 外键约束启用
node security-fixes/scripts/enable-foreign-keys.js
```

### 方法三：手动按文档修复

1. 阅读 `plans/` 目录下的修复计划文档
2. 按照文档中的步骤手动执行修复
3. 使用提供的代码示例和配置
