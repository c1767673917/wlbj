## 当前缓存架构

### 核心组件

1. **SmartCacheManager**: 智能缓存管理器

   - L1缓存: NodeCache（1000键，5分钟TTL）
   - L2缓存: LRU（5000项，30分钟TTL）

2. **CacheIntegration**: 缓存集成层

   - 业务缓存策略
   - 缓存失效管理
   - 性能监控

3. **CacheMiddleware**: 路由缓存中间件
   - API路由自动缓存
   - 条件缓存控制
   - 缓存键生成

### 缓存策略

- **订单数据**: 5分钟缓存
- **报价数据**: 3分钟缓存
- **供应商数据**: 30分钟缓存
- **用户数据**: 30分钟缓存
- **统计数据**: 15分钟缓存
