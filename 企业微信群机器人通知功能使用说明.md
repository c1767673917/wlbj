# 企业微信群机器人通知功能使用说明

## 功能概述

物流报价系统现已集成企业微信群机器人通知功能，当用户发布新订单时，系统会自动向配置了企业微信webhook的物流公司群发送通知消息。

## 功能特性

- ✅ 自动发送订单通知到企业微信群
- ✅ 支持Markdown格式的富文本消息
- ✅ 前端处理API调用，优化性能
- ✅ 完善的错误处理和重试机制
- ✅ 支持批量通知多个物流公司
- ✅ 灵活的webhook URL配置管理

## 使用步骤

### 1. 创建企业微信群机器人

1. 在企业微信中创建一个群聊
2. 在群聊中添加机器人：
   - 点击群聊右上角的"..."
   - 选择"群机器人"
   - 点击"添加机器人"
   - 设置机器人名称和头像
   - 复制生成的webhook URL

### 2. 配置物流公司的企业微信通知

#### 方法一：添加新物流公司时配置
1. 登录用户端：http://localhost:3000/user
2. 切换到"物流公司管理"标签页
3. 在"添加新物流公司"表单中：
   - 填写公司名称
   - 填写自定义链接名（可选）
   - 填写企业微信群机器人Webhook URL
4. 点击"确认添加"

#### 方法二：为现有物流公司配置
1. 在物流公司列表中找到目标公司
2. 在"企业微信通知"列中：
   - 如果显示"未配置"，点击"配置"按钮
   - 如果显示"✓ 已配置"，点击"编辑"按钮
3. 在弹出的对话框中输入webhook URL
4. 点击确认

### 3. 测试通知功能

1. 配置完成后，发布一个新订单
2. 系统会自动向所有配置了webhook的物流公司发送通知
3. 检查企业微信群是否收到通知消息

## Webhook URL格式

企业微信群机器人的webhook URL格式为：
```
https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key=YOUR_KEY
```

其中`YOUR_KEY`是企业微信为每个机器人生成的唯一标识。

## 通知消息格式

系统发送的通知消息包含以下信息：
- 📦 订单编号
- 🏢 发货仓库
- 📦 货物信息（截取前100字符）
- 📍 收货信息（截取前100字符）
- ⏰ 订单发布时间
- 💰 报价提醒

示例消息：
```
📦 **新订单报价提醒**

**订单编号**: ORD-20250126-001
**发货仓库**: 广州仓
**货物信息**: 清香牛肉579箱，香辣味1321箱，大骨牛肉500箱...
**收货信息**: 河南省漯河市临颍县繁阳路南侧经一路西侧...
**发布时间**: 2025/1/26 15:30:25

💰 **请及时登录系统提交报价！**
⏰ **通知时间**: 2025/1/26 15:30:25

---
*物流报价系统自动通知*
```

## 技术实现

### 前端优化
- 企业微信API调用在前端处理，减少服务器负载
- 实时验证webhook URL格式
- 用户友好的配置界面

### 后端支持
- 数据库自动迁移，添加`wechat_webhook_url`字段
- RESTful API支持webhook配置的增删改查
- 完善的错误处理和日志记录

### 安全性
- webhook URL格式验证
- 防止无效URL配置
- 错误信息不暴露敏感信息

## 故障排除

### 1. 通知未发送
- 检查webhook URL格式是否正确
- 确认企业微信群机器人是否正常工作
- 查看服务器日志中的错误信息

### 2. 消息格式异常
- 确认使用的是企业微信群机器人（不是个人微信）
- 检查webhook URL是否完整且有效

### 3. 配置保存失败
- 检查网络连接
- 确认webhook URL格式符合要求
- 查看浏览器控制台错误信息

## API接口

### 添加物流公司（支持webhook）
```http
POST /api/providers
Content-Type: application/json

{
  "name": "顺丰速运",
  "customAccessKey": "shunfeng",
  "wechatWebhookUrl": "https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key=YOUR_KEY"
}
```

### 更新webhook URL
```http
PUT /api/providers/:id/webhook
Content-Type: application/json

{
  "wechatWebhookUrl": "https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key=NEW_KEY"
}
```

## 注意事项

1. **频率限制**：企业微信群机器人有发送频率限制（20条/分钟），请合理使用
2. **URL安全**：请妥善保管webhook URL，避免泄露
3. **消息长度**：货物信息和收货信息会自动截取，避免消息过长
4. **网络依赖**：通知功能依赖网络连接，请确保服务器能访问企业微信API
